User Base Ops

a simple project for user signup, signin and password change.


RVM
 install rvm (ref: https://rvm.io/rvm/install)
 curl -L https://get.rvm.io | bash -s stable --ruby

Ruby 2.3.4
 rvm install ruby-2.3.4

Setup

- Clone user base ops repo:
 git clone https://github.com/misterwolf/user_base_ops.git

- Install required gems:
 gem install bundler && gem install foreman && bundle install

Start development server:
 rails s -p 3000
